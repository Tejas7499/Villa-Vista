Villa Vista is a full-stack web app inspired by Airbnb, built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. It allows users to explore, list, and book luxury villas seamlessly.

Key Features:

Add and manage villa listings

Search and filter properties by location or price

Secure user authentication (login/signup)

Real-time booking system

Responsive and modern UI

Tech Stack: HTML, CSS, JavaScript, Node.js, Express.js, MongoDB
